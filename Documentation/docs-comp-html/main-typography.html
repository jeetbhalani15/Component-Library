Example of Some Paragraph styles

You can use the mark tag to highlight text. Add text-mark class to mark tag To strike-through the text, add text-strike-through class
To make text bold (weight: 700), add text-bold-weight class
To make text regular weight (weight: 500), add text-regular-weight class
To make text light (weight: 300), add text-light-weight class
To make text small in size (12px), add text-small-size class
To make text color change there are 3 class, add one of the classes
primary-text-color, secondary-text-color, tertiary-text-color
